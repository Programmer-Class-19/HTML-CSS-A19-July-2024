HTML ( Hypertext Markup Language )

Dalam HTML terdapat serangkaian Markup yang akan kita sebut dengan TAG 
berfungsi untuk kasihtau / perintah browser bagaimana sebuah content akan di tampilkan,
misalnya saat kita ingin menampilkan sebuah text, text itu harus kita kasih tau 
kita mau buat apa misalnya mau buat paragraf , heding  judul , atau hyperlink

selain HTML TIM BERNERS LEE juga membuat : - HTTP
                                           - WWW 
                                           - Web Browser 
                                           - Web Server
                                           - Web Page 

HTML menciptakan sebuah struktur sebuah halaman dan menyajikan content
CSS digunakan untuk menghias halaman agar terlihat lebih bagus dan rapi 
Javascript  bisa digunakan banyak hal salah satunya untuk interaktivitas
