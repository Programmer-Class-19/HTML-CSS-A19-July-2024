text styling

color
memberi warna pada tulisan
  1. nama warna (blue)
  2. hexadecimal (#ff0000)
  3. rgb (rgb (255,0,0)
text-align
mengatur format paragraf / teks (kanan-kiri-tengah)
  1. left
  2. right
  3. center
  4. justify
text-indent
memberi identasi pada paragraf / teks
  1. px
  2. %
text-decoration
memberi dekorasi pada teks
  1. none
  2. underline
  3. overline
  4. line-through
text-transform
mengubah jenis huruf menjadi huruf besar, keil / kapital
  1. none (gada perubahan)
  2. lowercase (huruf kecil semua)
  3. uppercase (huruf besar semua)
  4. capitalize (awal kata menggunakan huruf kapital)
letter-spacing
mengatur spasi antar huruf
  1. normal
  2. px
word-spacing
mengatur spasi antar kata
  1. normal
  2. px
