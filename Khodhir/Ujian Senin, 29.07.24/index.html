Ujian HTML & CSS

Pentingnya HTML dan CSS dalam pengembangan web tidak bisa diabaikan...

Daftar Tidak Berurutan tentang HTML

  • HTML adalah fondasi dari semua halaman web.
  • HTML digunakan untuk membuat struktur halaman web.
  • HTML bekerja bersama dengan CSS dan JavaScript.

Daftar Berurutan tentang CSS

  1. CSS digunakan untuk mendesain halaman web.
  2. CSS menentukan tampilan elemen HTML.
  3. CSS dapat digunakan secara internal, eksternal, atau inline.
