Anatomi css
dalam Anatomi css terdapat struktur membentuk 
selector { property: value; }
contoh h1 { color: blue; } artinya 
selector adalah h1 
property adalah color:
value adalah blue; 
dan setiap css menggunaakan { kurung kurawal }

selector
> digunakan untuk memilih dan memanipilasi elemen spesifik pada HTML 
> elemen HTML dipilih berdasarkan tag , id , clas , bahkan pola / pattern 
> semakain kompleks stuktur HTML, selector juga semakin bisa kompleks /. spesifik
