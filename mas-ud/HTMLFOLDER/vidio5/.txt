pembahasan tag paragraf pada HTML 
ada beberapa tag yang akan kita bahas sebagaimana berikut :
tag <p> Paragraf </p> = untuk membuat paragraf 
tag <br> Line Break =  untuk mebuat garis baru
tag <hr> Rorizontal Rule  =  untuk membuat garis horizontal 
tag <b> Bold </b> =  / mempertebal font
tag <i> Italic </i> = untuk membuat teks bergaya miring 
tag <u> Underline </u> = untuk membuat di bawah teks ada garis 

kalau misalnya kita ingin membuat pada paragraf tersebut ada 
teks penting atau ada penekanan padanya kita bisa menggunakan tag ini 
tag <strong> Strong Emphasis </strong> dan <em> Emphasis </em>  
tag <strong> Strong Emphasis </strong> = untuk membuat fontnya tebal sama seperti tag <b> Bold </b>
tag <em> Emphasis </em> = untuk membuat gaya bteks miring sama seperti tag <i> italic </i>
