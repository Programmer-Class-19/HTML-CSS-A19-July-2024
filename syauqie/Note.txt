SENIN 22 JULY 2024 =============================================================

- HTML adalah bahasa pemrograman yang digunakan untuk membuat halaman web.
- HTML Singkatan dari HyperText Markup Language.
- HTML dibuat oleh Tim Berners-Lee.
- Tim Berners-Lee adalah seorang scientist di universitas Stanford, Inggris.
- Tim Berners-Lee termasuk salah satu pendiri dari World Wide Web.
- Link World Wide Web : https://en.wikipedia.org/wiki/World_Wide_Web
- Tim Berners-Lee Juga Bergabung di https://www.w3.org

- Web untuk melihat perkembangan teknologi web server di https://evolutionoftheweb.com

- HTML Ibarat Sebuah Manekin Tanpa Busana.
- Sedangkan CSS ibarat Busana yang di pakai oleh Manekin

- Dalam HTML mesti di awali dengan <!DOCTYPE html>
- Dalam HTML mesti ada Tag <html> dan <head> dan <body> di tutup dengan </body> dan </html>
- Contoh HTML :
<!DOCTYPE html> // HTML
<html> // Tag HTML
<head> // Tag Head
</head> // Tag Tutup Head
<body> // Tag Body
</body> // Tag Tutup Body
</html> // Tag Tutup HTML
- Setiap Tag pasti punya Tag Penutup
- Setiap tag pasti punya Atribut

MACAM MACAM TAG

- Judul Halaman : <title> - Menentukan judul halaman web yang akan ditampilkan di tab browser.
- CSS : <style> - Digunakan untuk menuliskan kode CSS yang akan mengatur tampilan halaman web.
- Link : <link> - Menghubungkan dokumen HTML dengan file eksternal seperti CSS atau ikon.
- Tabel : <table> - Membuat tabel untuk menampilkan data dalam bentuk baris dan kolom.
- Tabel Baris : <tr> - Menyatakan baris dalam sebuah tabel.
- Tabel Kolom : <td> - Menyatakan kolom dalam sebuah tabel.
- Paragraf : <p> - Menyatakan paragraf teks.
- Gambar : <img> - Menampilkan gambar dalam halaman web.
- Hyper Link : <a> - Membuat tautan hyperlink ke halaman web lain.
- Tombol : <button> - Membuat tombol interaktif yang dapat diklik oleh pengguna.
- Input : <input> - Menyediakan berbagai jenis input seperti teks, checkbox, radio button, dll.
- Dropdown : <select> - Membuat dropdown list untuk memilih satu opsi dari beberapa pilihan.
- List : <ul> dan <li> - Membuat daftar item dalam bentuk list.
- List Tabel : <table> dan <tr> dan <td> - Membuat daftar item dalam bentuk tabel.
- Numbering : <ol> dan <li> - Membuat daftar item berurutan dengan angka.
- Teks : <h1> dan <h2> dan <h3> dan <h4> dan <h5> dan <h6> dan <p> - Menyatakan teks dengan ukuran dan hierarki yang berbeda.
- Paragraf Teks : <br> dan <hr> - Menyisipkan baris baru atau garis horizontal dalam teks.
- Group Teks : <span> - Mengelompokkan teks untuk diberi gaya atau manipulasi tertentu.
- Grouping : <div> dan <span> - Mengelompokkan elemen HTML untuk diberi gaya atau manipulasi tertentu.
- Komentar : <!-- --> dan <!-- --> - Menyisipkan komentar yang tidak akan ditampilkan di halaman web.
- Form : <form> dan <input> dan <button> dan <select> dan <textarea> dan <label> dan <option> - Membuat formulir untuk pengguna memasukkan data.
- Script : <script> - Menyisipkan kode JavaScript dalam halaman web.
- Object : <object> dan <embed> dan <iframe> dan <applet> dan <param> dan <area> dan <map> - Menyisipkan objek multimedia seperti audio, video, atau aplikasi.
- Audio : <audio> dan <source> - Memasukkan elemen audio ke halaman web.
- Video : <video> dan <source> - Memasukkan elemen video ke halaman web.
